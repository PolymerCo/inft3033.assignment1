This application provides conversion tools for converting
between the units:

 Watts
 Kilowatts
 Metric Horsepower
 Electrical Horsepower
 Mechanical Horsepower

All conversions are bi-directional.

=========================

This application was developed using:
 
 Xcode 9.2 (9C40b)
 macOS High Sierra (10.13.6)
